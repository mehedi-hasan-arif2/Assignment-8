# 🎓 SkillSphere – Online Learning Platform

SkillSphere is a modern, responsive online learning platform built with Next.js. It allows users to explore skill-based courses, watch lessons, and manage their personal learning profiles with ease.

## 🔗 Live Link
- **Live URL:** https://skillsphere-mehedi-hasan-arif.netlify.app/

## 🎯 Project Purpose
The goal of this platform is to bridge the gap between industry experts and learners. It provides a structured environment where users can discover trending courses, search for specific skills, and securely manage their accounts to track their learning journey.

## 🚀 Key Features
- **Dynamic Search:** Real-time course search by title on the All Courses page.
- **Secure Authentication:** Integrated login, email registration, and Google Social Login.
- **Protected Routes:** Premium course details and static curriculum lists are locked for authenticated users only.
- **Profile Management:** Dynamic user profile page with live data and profile information update feature (Name & Image URL).
- **Responsive Layout:** Tailored layouts following strict breakpoints for Mobile, Tablet, and Desktop screens.
- **Interactive UX:** Smooth page animations and card hover effects using Framer Motion.

## 🛠️ NPM Packages Used
- `next` (Next.js App Router)
- `tailwindcss` (Styling framework)
- `daisyui` (UI components library)
- `better-auth` (Authentication and profile updates)
- `framer-motion` (Smooth interface animations)
- `swiper` (Modern touch slider/carousel component)
- `react-hot-toast` & `sonner` (Toast notifications for user alerts)
- `lucide-react` (Clean UI icons)
