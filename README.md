# SkillSphere - Online Learning Platform

SkillSphere is a modern, responsive online learning platform built with Next.js. It allows users to explore various professional courses, view detailed curriculum, and manage their personal learning profile.

### Key Features
* **Modern UI/UX:** Built with Tailwind CSS and styled for a unique professional look.
* **Course Discovery:** Explore courses with real-time search functionality.
* **Protected Routes:** Secure navigation ensuring only authenticated users can access course details.
* **Authentication:** User login and registration system with Google Auth support.
* **Profile Management:** Users can update their profile information (Name & Image URL).
* **Smooth Animations:** Implemented using **Framer Motion** for a seamless user experience.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.

### Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Animation:** Framer Motion
* **Authentication:** BetterAuth
